# MVP Scope

## 1. 문서 목적
본 문서는 원격 데스크톱 연결 프로그램의 초기 출시 버전(MVP)에 포함할 기능과 제외할 기능을 명확히 정의하여 개발 범위를 통제하기 위한 문서이다.

## 2. MVP 목표
- 사용자가 설치 후 빠르게 원격 PC에 연결할 수 있어야 한다.
- 기본적인 원격 제어 경험이 안정적으로 동작해야 한다.
- 보안상 최소 기준인 승인, 인증, 암호화, 로그가 포함되어야 한다.
- 실제 사용자 테스트가 가능한 수준의 연결 흐름과 핵심 기능을 제공해야 한다.

## 3. MVP 포함 범위
### 필수 기능
- 장치 등록 및 장치 식별
- 원격 연결 요청 및 승인
- 원격 화면 보기
- 마우스 및 키보드 입력 제어
- 세션 시작, 종료, 실패 상태 표시
- 기본 접속 로그 저장
- 자동 재연결
- 기본 오류 메시지 및 복구 가이드

### 조건부 포함 기능
- 최근 접속 목록
- 즐겨찾기 장치 저장
- 클립보드 텍스트 공유
- 단일 파일 또는 소용량 파일 전송
- 로컬 드라이브 탐색 기반 리디렉션

### MVP 대상 플랫폼
- Host: Windows
- Client: Windows

## 4. MVP 제외 범위
- 조직 관리자 콘솔
- 다중 사용자 동시 접속
- 모바일 앱
- 세션 녹화
- 음성/채팅 기능
- 고급 권한 정책
- 대용량 파일 전송 최적화
- 다중 모니터 고급 제어
- macOS 및 Linux 지원

## 5. MVP 성공 기준
- 내부 테스트 환경에서 연결, 승인, 제어 흐름이 안정적으로 재현될 것
- 일반 사무용 네트워크 환경에서 기본 조작이 가능할 수준의 반응성을 확보할 것
- 주요 오류 상황에서 앱이 비정상 종료되지 않고 사용자에게 복구 방법을 안내할 것
- 최소 1차 사용자 시나리오 3종 이상을 end-to-end로 수행할 수 있을 것

## 6. 우선순위
### P0
- 장치 등록
- 연결 요청/승인
- 화면 전송
- 입력 제어
- 암호화
- 기본 로그

### P1
- 자동 재연결
- 최근 접속 목록
- 즐겨찾기
- 클립보드 텍스트 공유
- 기본 감사 로그 영구 저장

### P2
- 파일 전송
- 다중 모니터 전환
- 세션 상태 상세 정보
- 로컬 드라이브 리디렉션 고도화

## 7. MVP 이후 확장 후보
- 무인 접속
- 계정 기반 로그인
- 조직 단위 장치 관리
- 고급 파일 전송
- 세션 녹화 및 감사 기능
- 멀티 플랫폼 지원
